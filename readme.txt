== Description ==
### 一款精美的在线 旅行地图标记,记录日志插件!
- [Demo / 演示地址](https://wpapi.aliluv.cn/maps)
`帮助用户记录旅途中经过的每一个地点，点亮足迹地图，让你可以清晰的看到自己去过哪些地方，还可以储存你在旅途中拍摄的照片，和别人分享你美好的行程!`


== Installation ==
1. 上传 `jiutu-mapmarker`目录 到 `/wp-content/plugins/` 目录或后台搜索 `地图标记`
2. 在后台插件菜单激活该插件
3. 演示地址:https://wpapi.aliluv.cn/maps
